MDReport (쐐기돌보고서) 1.1.0

사용 가능한 명령어 정리
!내돌 - 내돌을 링크함 (명령어를 입력한 채널로)  (본인전용)
!돌,!쐐기돌,!모든돌,!전체돌 - 다른사람과 나의 돌을 모두 요청함

!주차 - 주차못한캐릭이 남은경우 그캐릭의 정보를, 모두 했을경우 주차범위를 출력

!직업이름 - 해당 직업의 돌이 있으면 링크
!던전이름  - 해당 던전의 돌이 있으면 링크

!속성, 이번주, 지난주,다음주,지지난주,다다음주속성 = 해당 주간의 속성 출력. (본인전용)

!버전!버젼! - 버전체크 (본인전용)

1.0.21+
!지금돌,!돌내놔 - 현재 "접속"한 캐릭터 돌만 링크 (애드온을 설치한 사람 모두)
!지금내돌,!무슨돌- 현재 접속한 캐릭터의 돌 을 "나만" 링크  (본인전용)

1.0.23+
!아무+단수 - 아무 던전이든 해당 단수의 돌이 있으면 링크
!던전이름+단수 - 해당 던전의 해당 단수의 돌이 있으면 링크

1.0.25+
!전부,웅심,질병,저주,독,천,판금,사슬,가죽,방패 등등 직업 특징으로 검색시 해당 직업군 중에 돌이 있으면 링크

1.1.0+
무기종류 검색기능 추가 - 검색 장비를 드랍하는 던전의 돌이 있을 경우 링크

무기종류는 도검, 단검, 둔기,방패,총 등을 말하고
무기유형은 원거리, 근접, 한손 처럼 범주를 뜻합니다.

!무기종류 (ex. !한손도검, !단검, !전투검) 
!주능력치!무기종류 (ex. !힘!한손도검, !민첩!단검, !민첩!지팡이)

!주능력치!무기유형(=양손장비, 한손장비, 근접장비, 원거리장비)
무기유형은 !주능력치 종류와 함께 입력해야만 검색이 가능합니다
(ex. !힘!양손장비 - 착용가능여부와 상관없이 힘이 붙은 모든 양손장비를 검색)
(ex. !민첩!원거리 - 총,활,석궁을 모두 검색)

해당 전문화가 착용가능한 무기를 드랍하는 던전을 검색하고 싶은 경우에는 
!전문화!무기종류
!전문화!무기유형
으로 검색할 수 있습니다. !회드!양손장비 처럼 무기 유형을 지정하거나,  !회드!단검  특정 장비를 지정할 수도 있습니다

1.1.2+
별도의 조합없이 !숫자로 특정 단수를 검색할 수 있습니다.
(ex. !15, !17)
또는 이상(+), 이하(-), 미만, 초과와 결합하여 검색할수도 있습니다.
(ex. !15이하, !19+, !12-, !10초과)
혹은 "~"과 함께 레벨 범위를 지정해서 검색할 수 있습니다
(ex. !15~16,  !18~19)
